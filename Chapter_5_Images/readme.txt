<img ... /> - no closing tag because it's an empty element
src="..."   - relative path of image
alt="..."   - text description for blind people or when the image isn't found for any reason
title="..." - usually only shown to users that hover over an image; also read by by screen reader software

OLD CODE align:

align attribute in <img \> was used to indicate how text should flow around an image. Not used in HTML5 (CSS)
HORIZONTAL:align="right" or align="left"
VERTICAL:align="top" or align="bottom" or align="middle"
